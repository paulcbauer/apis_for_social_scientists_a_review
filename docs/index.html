Preface

Permanent link: https://paulcbauer.github.io/apis_for_social_scientists_a_review/

The present online book provides a review of APIs that may be useful for social scientists. Please start by reading the Introduction. The material was/is being developed by various contributors, which you can find above and in the contributor section of the corresponding github repository. If you are interested in contributing, please check out the Section How to contribute in the github README.
